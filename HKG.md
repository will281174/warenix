# Introduction #
一個為 CD-ROM 而製的的高登 app.

https://play.google.com/store/apps/details?id=org.dyndns.warenix.hkg

Blog: http://warenix.xanga.com/tags/hkg

# 功能 #

  * 接駁到反應較平常 forums 快的 http://m.hkgolden.com
  * 以暗灰色顯示區分獲負評的題目
  * 簡化分頁，只有回到首頁與下一頁
  * 轉台
  * 賣飛佛
  * 作為預設 APP 開 URL
  * 利用 GOOGLE 做搜尋功能 (有 QUOTA 限制)
  * 相集模式
  * 直接輸入 POST ID
  * 記下每個 POST 的最後閱讀頁數
